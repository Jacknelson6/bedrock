| to                          | object   | { opacity: 1, y: 0 }  |
| className                   | string   | —              |
| textAlign                   | string   | —              |
| threshold                   | number   | 0.1            |
| rootMargin                  | string   | "-100px"       |
| onLetterAnimationComplete   | function | —              |

Usage:
```jsx
<SplitText
  text="Hello World!"
  duration={0.6}
  delay={100}
  ease="power3.out"
  splitType="chars"

  to={{ opacity: 1, y: 0 }}
```

################################################################################
# INSTALLATION
################################################################################

Install via CLI (recommended):
  npx jsrepo add github/DavidHDev/react-bits/<component-name>

Or copy-paste from https://reactbits.dev

Each component page at reactbits.dev includes:
  - Live interactive demo
  - Copy-paste source code in 4 variants (JS-CSS, JS-TW, TS-CSS, TS-TW)
  - Prop table and usage examples

Common dependencies used across components:
  - motion (Framer Motion)
  - gsap + @gsap/react (GreenSock)
  - three + @react-three/fiber + @react-three/drei (Three.js)
  - ogl (lightweight WebGL)
  - matter-js (physics)
  - lenis (smooth scroll)
  - gl-matrix (math)
  - lucide-react (icons)
  - postprocessing / @react-three/postprocessing
