## Accordion

Description: A vertically stacked set of collapsible containers allowing users to toggle content visibility.

Installation:
  npx motion-primitives@latest add accordion

### Accordion (root)

Props:
  children          ReactNode                                    The content elements of the accordion
  className         string                                       Optional CSS class for styling the accordion container
  transition        Transition                                   Transition settings from motion for animation effects
  variants          { expanded: Variant; collapsed: Variant }    Custom variants for expanded and collapsed states
  expandedValue     React.Key | null                             Controlled expanded value
  onValueChange     (value: React.Key | null) => void            Callback function when the expanded value changes

### AccordionItem

Props:
  value             React.Key       Unique identifier for controlling the accordion item state
  children          ReactNode       The content elements of the accordion item
  className         string          Optional CSS class for styling the accordion item container

### AccordionTrigger

Props:
  children          ReactNode       Content of the trigger, typically a string or formatted text
  className         string          Optional CSS class for styling the trigger element

### AccordionContent

Props:
  children          ReactNode       The content displayed when the accordion item is expanded
  className         string          Optional CSS class for styling the content container

Usage Example:

```jsx
import {
  Accordion,
  AccordionItem,
  AccordionTrigger,
  AccordionContent,
} from '@/components/core/accordion';

export function AccordionBasic() {
  return (
    <Accordion className='flex w-full flex-col divide-y divide-zinc-200 dark:divide-zinc-700'>
      <AccordionItem value='getting-started'>
        <AccordionTrigger className='w-full py-0.5 text-left text-zinc-950 dark:text-zinc-50'>
          Getting Started
        </AccordionTrigger>
        <AccordionContent>
          <p className='text-zinc-500 dark:text-zinc-400'>
            Discover the fundamental concepts of Motion-Primitives...
          </p>
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  );
}
```

Notes:
  - Icon rotation: use group-data-[expanded]:... utilities on trigger children
  - Custom variants: define custom animation variants for expanded/collapsed states
  - Spring transitions: supports spring-based animations with stiffness and damping
  - Controlled state: manage expanded state externally via expandedValue and onValueChange

## AnimatedBackground

Description: Visually highlights selected items by sliding a background into view when hovered over or clicked. Ideal for interactive lists, menus, or navigations where clear selection feedback is important.

Installation:
  npx motion-primitives@latest add animated-background

### AnimatedBackground

Props:
  children          ReactElement[] (each must have data-id string attribute)   Content to display; each child must have a unique data-id attribute
  defaultValue      string                                                      The default value used as the active item identifier
  onValueChange     (newActiveId: string | null) => void                        Callback triggered when the active item changes
  className         string                                                      CSS class name for custom styling of the animated background
  transition        Transition                                                  Motion transition effects from motion/react
  enableHover       boolean (default: false)                                    Enables hover-based activation; when true, spacing between children cannot be added

Notes:
  - Children must include data-id attribute for tracking active state
  - Active items receive data-checked="true" attribute for styling
  - Uses layoutId for smooth morphing animations between selections
  - Works with tabs, segmented controls, card grids, and navigation menus

## AnimatedGroup

Description: A wrapper that adds animated transitions to a group of child elements. Perfect for creating staggered animations for lists, grids, or any collection of components.

Installation:
  npx motion-primitives@latest add animated-group

### AnimatedGroup

Props:
  children          ReactNode                                           The child elements to be animated
  className         string                                              Optional CSS class for styling
  variants          { container?: Variants; item?: Variants }           Custom variants for container and item animations
  preset            'fade' | 'slide' | 'scale' | 'blur-sm' | 'blur-slide'   Preset animations to apply
  as                React.ElementType (default: 'div')                  HTML element to render the component as
  asChild           React.ElementType (default: 'div')                  HTML element to render child elements as

Available Presets: fade, slide, scale, blur-sm, blur-slide, zoom, flip, bounce, rotate, swing

Usage Example (basic preset):

```jsx
import { AnimatedGroup } from '@/components/core/animated-group';

<AnimatedGroup preset="scale" className="grid grid-cols-3 gap-4">
  <img src="/img1.jpg" />
  <img src="/img2.jpg" />
  <img src="/img3.jpg" />
</AnimatedGroup>
```

Usage Example (custom variants):

```jsx
<AnimatedGroup
  variants={{
    container: {
      hidden: { opacity: 0 },
      visible: { opacity: 1, transition: { staggerChildren: 0.05 } },
    },
    item: {
      hidden: { opacity: 0, y: 40, filter: 'blur(4px)' },
      visible: { opacity: 1, y: 0, filter: 'blur(0px)' },
    },
  }}
>
  {children}
</AnimatedGroup>
```

## Carousel

Description: A flexible, customizable carousel component with navigation controls and indicators for cycling through content items.

Installation:
  npx motion-primitives@latest add carousel

### Carousel (root)

Props:
  children          ReactNode                              Content elements of the carousel
  className         string                                 Optional CSS class for styling
  initialIndex      number (default: 0)                    Starting position of active item
  index             number                                 Controlled index of active carousel item
  onIndexChange     (newIndex: number) => void             Callback when the active index changes
  disableDrag       boolean (default: false)               Disables dragging functionality

### CarouselContent

Props:
  children          ReactNode       The sliding elements or items of the carousel
  className         string          Optional CSS class for additional styling
  transition        Transition      Optional motion transition for animating slides (default: spring preset)

### CarouselItem

Props:
  children          ReactNode       The individual content item within the carousel
  className         string          Optional CSS class for styling the carousel item

### CarouselNavigation

Props:
  className         string                      Optional CSS class for styling the navigation container
  classNameButton   string                      Optional CSS class for styling the navigation buttons
  alwaysShow        boolean (default: false)    Whether navigation buttons are always visible

### CarouselIndicator

Props:
  className         string      Optional CSS class for styling the indicator container
  classNameButton   string      Optional CSS class for styling the indicator buttons

### useCarousel hook

Returns:
  index             number                              The current index of the active carousel item
  setIndex          (newIndex: number) => void           Function to update the index
  itemsCount        number                              Total number of items in the carousel
  setItemsCount     (newItemsCount: number) => void     Function to set the number of items

```jsx
import {

    </CarouselItem>
```

## Cursor

Description: A custom cursor component with optional spring animations. It can be globally applied to the page or attached specifically to a parent element.

Installation:
  npx motion-primitives@latest add cursor

### Cursor

Props:
  children          ReactNode                                  Content rendered within the custom cursor
  className         string                                     Optional CSS class for styling the cursor container
  springConfig      SpringOptions                              Configuration for spring physics governing cursor movement
  attachToParent    boolean (default: false)                   When true, cursor visibility limited to parent hover state
  transition        Transition                                 Animation transition settings from motion library
  variants          { initial; animate; exit }                 Animation states
  onPositionChange  (x: number, y: number) => void            Callback triggered when cursor position changes

Usage Example:

```jsx
import { Cursor } from '@/components/core/cursor';

<div className='relative'>
  <Cursor
    attachToParent
    variants={{
      initial: { height: 0, opacity: 0, scale: 0.3 },
      animate: { height: 'auto', opacity: 1, scale: 1 },
      exit: { height: 0, opacity: 0, scale: 0.3 },
    }}
    transition={{ type: 'spring', duration: 0.3, bounce: 0.1 }}
    springConfig={{ bounce: 0.001 }}
  >
    <div>Custom cursor content</div>
  </Cursor>
  <img src='/photo.jpg' alt='Photo' />
</div>
```

Props:
  children          ReactNode       Content of the trigger
  className         string          CSS class

### DialogContent

Props:
  children          ReactNode                          Content of the dialog

### DialogHeader

  children          ReactNode       Title and description container
  className         string          CSS class

Props:
  children          ReactNode       Heading content
  className         string          CSS class

### DialogDescription

Props:
  children          ReactNode       Description content
  className         string          CSS class

### DialogClose

Props:
  className         string          CSS class
  children          ReactNode       Custom close content (defaults to X icon)
  disabled          boolean         Whether close is disabled

```jsx
import {
```

```jsx
const [isOpen, setIsOpen] = useState(false);

```

Usage Example (custom animation):


## InView

Description: Easily animate elements when they come into view. Apply animations to elements when they enter the viewport or when they are fully visible.

Installation:
  npx motion-primitives@latest add in-view

### InView

Props:
  children          ReactNode                                    The content inside the InView component
  variants          { hidden: Variant; visible: Variant }        Variants to define the animation states
                    (default: { hidden: { opacity: 0 }, visible: { opacity: 1 } })
  transition        Transition                                   Specifies the animation transitions
  viewOptions       UseInViewOptions                             Options to configure the in-view behavior
  as                React.ElementType (default: 'div')           The HTML element to render the component as
  once              boolean (default: false)                     If true, the animation will play only once

Usage Example (basic):

```jsx
import { InView } from '@/components/core/in-view';

<InView
  variants={{
    hidden: { opacity: 0, y: 100, filter: 'blur(4px)' },
    visible: { opacity: 1, y: 0, filter: 'blur(0px)' },
  }}
  viewOptions={{ margin: '0px 0px -200px 0px' }}
  transition={{ duration: 0.3, ease: 'easeInOut' }}
>
  <div className='max-w-96'>
    <p>Content to animate</p>
  </div>
</InView>
```

Usage Example (staggered children, animate once):

```jsx
<InView
  viewOptions={{ once: true, margin: '0px 0px -250px 0px' }}
  variants={{
    hidden: { opacity: 0 },
    visible: {
      opacity: 1,
      transition: { staggerChildren: 0.09 },
    },
  }}
>
  <div className='columns-2 gap-4'>
    {images.map((img, index) => (
      <motion.div
        variants={{
          hidden: { opacity: 0, scale: 0.8, filter: 'blur(10px)' },
          visible: { opacity: 1, scale: 1, filter: 'blur(0px)' },
        }}
        key={index}
      >
        <img src={img} />
      </motion.div>
    ))}
  </div>
</InView>
```

## TransitionPanel

Description: Easy way to switch between different pieces of content with enter and exit animations. Designed for onboarding cards, settings adjustments, and interactive content.

Installation:
  npx motion-primitives@latest add transition-panel

### TransitionPanel

Props:
  children          React.ReactNode[]       The content that the panel will transition through
  className         string                  Optional CSS class for the panel container styling
  transition        Transition              Motion transition settings controlling animation effects
  activeIndex       number                  Index of the currently active child element to display
  variants          Object                  Animation variants with enter, center, and exit properties
  ...motionProps    MotionProps             Additional motion props spread to the motion.div component

Variants Structure:
  enter   - Initial animation state when content appears
  center  - Target/active state when content is displayed
  exit    - Final animation state when content disappears

Usage Example:

```jsx
import { TransitionPanel } from '@/components/core/transition-panel';

const [activeIndex, setActiveIndex] = useState(0);

<TransitionPanel
  activeIndex={activeIndex}
  transition={{ duration: 0.2, ease: 'easeInOut' }}
  variants={{
    enter: { opacity: 0, y: -50, filter: 'blur(4px)' },
    center: { opacity: 1, y: 0, filter: 'blur(0px)' },
    exit: { opacity: 0, y: 50, filter: 'blur(4px)' },
  }}
>
  <div>Panel 1 content</div>
  <div>Panel 2 content</div>
  <div>Panel 3 content</div>
</TransitionPanel>
```

Notes:
  - Supports directional animations via the custom prop
  - Works with spring physics and duration-based transitions
  - Integrates with AnimatePresence for lifecycle management

## TextEffect

Description: Easily animate text content with various effects. Apply animations per character, per word, or per line, and customize using custom variants or preset animations.

Installation:
  npx motion-primitives@latest add text-effect

### TextEffect

Props:
  children                  string                     The text content to be animated (required)
  per                       'word' | 'char' | 'line' (default: 'word')   Animation granularity
  as                        keyof JSX.IntrinsicElements (default: 'p')   HTML tag to render
  variants                  { container?: Variants; item?: Variants }    Custom Framer Motion variants
  className                 string                     CSS classes for styling
  preset                    'blur-sm' | 'fade-in-blur' | 'scale' | 'fade' | 'slide' (default: 'fade')   Built-in animation presets
  delay                     number (default: 0)        Initial delay in seconds before animation starts
  speedReveal               number (default: 1)        Controls reveal animation speed (higher = faster)
  speedSegment              number (default: 1)        Controls individual segment animation speed
  trigger                   boolean (default: true)    Whether animation executes
  onAnimationComplete       () => void                 Callback fired when animation finishes
  onAnimationStart          () => void                 Callback fired when animation begins
  segmentWrapperClassName   string                     CSS classes for segment wrapper elements
  containerTransition       Transition                 Custom transition for container
  segmentTransition         Transition                 Custom transition for segments
  style                     CSSProperties              Inline styles applied to component

Usage Example (per character):

```jsx
import { TextEffect } from '@/components/core/text-effect';

<TextEffect per='char' preset='fade'>
  Animate your ideas with motion-primitives
</TextEffect>
```

Usage Example (per word with blur):

```jsx
<TextEffect per='word' as='h3' preset='blur'>
  Animate your ideas with motion-primitives
</TextEffect>
```

Usage Example (custom variants):

```jsx
const fancyVariants = {
  container: {
    hidden: { opacity: 0 },
    visible: { opacity: 1, transition: { staggerChildren: 0.05 } },
  },
  item: {
    hidden: { opacity: 0, y: Math.random() * 100 - 50, rotate: Math.random() * 90 - 45, scale: 0.3 },
    visible: { opacity: 1, y: 0, rotate: 0, scale: 1, transition: { type: 'spring', damping: 12, stiffness: 200 } },
  },
};

<TextEffect per='word' variants={fancyVariants}>
  Animate your ideas with motion-primitives
</TextEffect>
```

Usage Example (speed control):

```jsx
<TextEffect preset='fade-in-blur' speedReveal={1.1} speedSegment={0.3}>
  Animate your ideas with motion-primitives.
</TextEffect>
```

## ProgressiveBlur

Description: A Progressive Blur component that creates a layered blur effect using motion and gradient masks. It progressively blurs layers based on direction and intensity, adding visual depth.

Installation:
  npx motion-primitives@latest add progressive-blur

### ProgressiveBlur

Props:
  direction         'top' | 'right' | 'bottom' | 'left' (default: 'top')   The direction of the blur effect
  blurLayers        number (default: 8)                                      The number of blur layers
  className         string (default: '')                                     CSS class for the component
  blurIntensity     number (default: 0.25)                                   The intensity of the blur effect

Also accepts HTMLMotionProps<'div'> for motion animations.

Usage Example (basic):

```jsx
import { ProgressiveBlur } from '@/components/core/progressive-blur';

<div className='relative'>
  <img src='/photo.jpg' alt='Photo' />
  <ProgressiveBlur
    direction='bottom'
    blurIntensity={0.5}
    className='absolute inset-0'
  />
</div>
```

Usage Example (hover interaction):

```jsx
const [isHovered, setIsHovered] = useState(false);

<div
  onMouseEnter={() => setIsHovered(true)}
  onMouseLeave={() => setIsHovered(false)}
>
  <img src='/photo.jpg' alt='Photo' />
  {isHovered && (
    <ProgressiveBlur direction='bottom' blurIntensity={1} />
  )}
</div>
```

Usage Example (combined with InfiniteSlider):

```jsx
<div className='relative'>
  <InfiniteSlider>
    <img src='/item1.jpg' />
    <img src='/item2.jpg' />
  </InfiniteSlider>
  <ProgressiveBlur direction='left' blurIntensity={1} className='absolute left-0 top-0 h-full w-24' />
  <ProgressiveBlur direction='right' blurIntensity={1} className='absolute right-0 top-0 h-full w-24' />
</div>
```

Notes:
  - Uses linear gradient masks with calculated angle based on direction
  - Backdrop filter for progressive blur application
  - Mask image technique for gradient-based visibility control
