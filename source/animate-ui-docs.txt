================================================================================
INSTALLATION
================================================================================

Animate UI uses the shadcn/ui installation methodology:
1. Initialize shadcn/ui (run init command)
2. Add components via CLI (e.g., npx shadcn add ...)
3. Import and use components

Example:
```tsx
import { SlidingNumber } from '@/components/animate-ui/primitives/texts/sliding-number';

export default function Home() {
  return (
    <div>
      <SlidingNumber />
    </div>
  );
}
```

- "path": Animates pathLength from 0 to 1
- "path-loop": Animates pathLength from 1 to 0 to 1
- "default": Icon-specific customized animation

Usage:
```jsx
<ArrowRight animation="pointing" />
```

--------------------------------------------------------------------------------

## Icon Triggers
Path: /docs/icons/usage/triggers

| Prop               | Type             | Default | Description                         |
|--------------------|------------------|---------|-------------------------------------|
| animate?           | boolean | string  | false   | Sets animate state                  |
| animateOnHover?    | boolean | string  | false   | Animate on hover                    |
| animateOnTap?      | boolean | string  | false   | Animate on tap                      |
| animateOnView?     | boolean | string  | false   | Animate when in viewport            |
| animateOnViewMargin?| string          | "0px"   | Viewport margin for inView trigger   |
| animateOnViewOnce? | boolean          | true    | Animate only once on first view      |

Note: animateOnTap returns to initial state when click released; use completeOnStop to ensure animations finish.

--------------------------------------------------------------------------------

## Icon Persistence
Path: /docs/icons/usage/persistence

| Prop                 | Type    | Default | Description                                      |
|----------------------|---------|---------|--------------------------------------------------|
| persistOnAnimateEnd? | boolean | false   | Remain in animate state (no return to initial)    |
| initialOnAnimateEnd? | boolean | false   | Set initial state without transition at end        |
| completeOnStop?      | boolean | false   | Finish current animation before returning to initial |

--------------------------------------------------------------------------------

## Icon Timing
Path: /docs/icons/usage/timing

| Prop       | Type    | Default | Description                          |
|------------|---------|---------|--------------------------------------|
| delay?     | number  | 0       | Milliseconds before animation starts |
| loop?      | boolean | false   | Repeat animation continuously        |
| loopDelay? | number  | 0       | Milliseconds between loop cycles     |
