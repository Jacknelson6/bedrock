# Toolbars & Menus — Bedrock Reference

| Need | Component |
|------|-----------|
| macOS dock | **Dock** (ReactBits) — proximity-scaling icon dock |
| Gooey navigation | **GooeyNav** (ReactBits) — gooey liquid nav |
| Pill navigation | **PillNav** (ReactBits) — pill-shaped nav tabs |
| Card navigation | **CardNav** (ReactBits) — card-based nav |
| Bubble menu | **BubbleMenu** (ReactBits) — floating bubble menu |
| Staggered menu | **StaggeredMenu** (ReactBits) — staggered reveal menu |
| Flowing menu | **FlowingMenu** (ReactBits) — flowing nav links |
| Infinite menu | **InfiniteMenu** (ReactBits) — infinite scroll menu |
| Glass icons | **GlassIcons** (ReactBits) — frosted glass icon buttons |

**For modals/dialogs/popovers**, use your UI framework (shadcn/ui Dialog, Radix Popover, etc.) with motion/react for animation. Bedrock focuses on unique interactive components, not standard dialog patterns.
