# Toolbars & Menus — Bedrock Reference

| Need | Component |
|------|-----------|
| Floating context toolbar | **ToolbarDynamic** ⭐ (Motion Primitives) |
| Expanding toolbar | **ToolbarExpandable** (Motion Primitives) |
| Dialog from trigger | **MorphingDialog** (Motion Primitives) — layout animation morph |
| Popover from trigger | **MorphingPopover** (Motion Primitives) |
| Dropdown menu | **BasicDropdown** (SmoothUI) — staggered items |
| Modal overlay | **BasicModal** (SmoothUI) — spring entrance |
| Toast notification | **BasicToast** (SmoothUI) — slide + auto-dismiss |
| Rich popover | **RichPopover** (SmoothUI) |
| Liquid popover | **GooeyPopover** (SmoothUI) — SVG gooey filter |
| Dynamic island | **DynamicIsland** (SmoothUI) — iOS-style morph |

**MorphingDialog** is unique — creates strong spatial continuity. The user sees where the content came from and where it goes back to.
